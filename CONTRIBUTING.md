# Contributing to Profolio AI

First off, thank you for considering contributing to Profolio AI! It's people like you that make Profolio AI such a great tool.

## How Can I Contribute?

### 🐛 Reporting Bugs
This section guides you through submitting a bug report for Profolio AI. Following these guidelines helps maintainers and the community understand your report, reproduce the behavior, and find related reports.

- Use the **Bug Report Issue Template** provided.
- Explain the problem and include additional details to help maintainers reproduce the problem:
  - **Use a clear and descriptive title** for the issue to identify the problem.
  - **Describe the exact steps which reproduce the problem** in as many details as possible.
  - **Provide specific examples to demonstrate the steps.**

### ✨ Suggesting Enhancements
This section guides you through submitting an enhancement suggestion for Profolio AI, including completely new features and minor improvements to existing functionality.

- Use the **Feature Request Issue Template** provided.
- Provide a step-by-step description of the suggested enhancement in as many details as possible.

### 💻 Pull Requests
The process described here has several goals:
- Maintain Profolio AI's quality
- Fix problems that are important to users
- Engage the community in working toward the best possible Profolio AI
- Enable a sustainable system for Profolio AI's maintainers to review contributions

Please follow these steps to have your contribution considered by the maintainers:
1. Fork the repo and create your branch from `main`.
2. Ensure you have tested your code thoroughly and it does not break existing functionality.
3. Keep your commits atomic, and use conventional commit formats (e.g. `feat: add new button`, `fix: correct typo in hero section`).
4. Issue a Pull Request utilizing the mandatory template formatting.

## Development Setup
Check the root `README.md` for a comprehensive step-by-step installation guide via Docker or directly on the host using Spring Boot and NodeJS.
